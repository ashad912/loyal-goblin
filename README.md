- server/src/routes/admin.js
- server/src/loaders
- client/src/components/tabs/events

## Run server: 
    1. Install mongoDB server
    2. Open terminal, run: 'cd server'
    3. Run: npm i
    4. Run: npm run dev-noenv
## Run client (user): 
    1. Open terminal, run: 'cd client'
    2. Run: npm i
    3. Run: npm start
## Run admin-client:
    1. Open terminal, run: 'cd admin'
    2. Run: npm i
    3. Run: npm start
